Detailed Description

Public class Clerk.cs. Part of the ACHClerk namespace, which represents a program capbable delivering an easy way for employees of the bank to provide ACH transition packets to customers.

The Clerk represents the main brain of the ACH packet operation. It is what handles all document manipulation, conversion, presentation, etc.

It is realized, as of v1.0, by ClerkForm.cs.

Author: Tyler Vanover. Created: 2014-06-26. Version: 1.0.
