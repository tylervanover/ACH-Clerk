Detailed Description

Public class PacketEntry.cs. Part of the ACHClerk namespace, which represents a program capbable delivering an easy way for employees of the bank to provide ACH transition packets to customers.

A PacketEntry is simply a collection of features regarding a single PDF document. Each PacketEntry has a unique ID, a company name, and a list of tags – the list of tags will the user to type in a string in the search bar and have a number of documents show as a result.

Author: Tyler Vanover. Created: 2014-06-26. Version: 1.0.
