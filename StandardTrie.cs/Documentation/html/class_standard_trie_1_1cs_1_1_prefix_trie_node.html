Detailed Description

Author: Tyler Vanover Created: 2014-07-04 Version: 1.0

Part of StandardTrie.cs namespace.

A node of a prefix tree. The prefix tree will allow the user to build a structure of characters such that they can perform traversals of complete words, and of prefixes to words during a search.

bool StandardTrie.cs.PrefixTrieNode.EndOfPath ( )

Determines if this node is the end of a complete path. What this means is that even though a valid word may end at this node, there could be subsequent nodes representing additional words. "flower" and "flowers" is a prime example.

bool StandardTrie.cs.PrefixTrieNode.IsEnd
getset

True/False value indicating if this node is the end of a valid word in the inserted set. May not reflect the end of a search path, however, as an example of inserting "flower" and then "flowers" would mean that both this 'r'-child and this 'r'-child's 's'-child would be marked IsEnd, as they are valid words. Only if all children are null is this the true end of a search path.
