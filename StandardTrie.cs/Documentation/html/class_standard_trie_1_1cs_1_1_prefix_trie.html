Detailed Description

Author: Tyler Vanover Created: 2014-07-04 Version: 1.0

Part of StandardTrie.cs namespace.

The PrefixTrie structure. This class will handle the manipulation of nodes within the trie. It has a single root node, from which the rest of the insertion, contains, and removal methods can be initiated.
